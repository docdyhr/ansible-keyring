# Code of Conduct

## Our Standard

This project expects respectful, professional, and constructive collaboration.

Examples of behavior that contribute to a healthy environment:

- Using clear and respectful language
- Focusing feedback on code, behavior, and outcomes
- Accepting constructive criticism in good faith
- Helping others learn the project conventions
- Respecting differing experience levels and viewpoints

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Discriminatory or exclusionary language
- Publishing private information without permission
- Deliberately disruptive or bad-faith interactions

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project interactions.

## Reporting

If you encounter unacceptable behavior, contact the maintainer through a private channel instead of escalating publicly.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.
